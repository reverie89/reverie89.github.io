vtsuki.net revamp

Year: 2025  |  hobby

vtsuki.net is rebuit and now powered by Hugo and agentic AI prompts. The site features a modern design, theme switching, mobile responsiveness, and advanced automation. Japanese and Singlish content are AI-generated, and the site is deployed via GitHub Pages.

  • Powered by Hugo to generate the site
  • Agentic AI prompts via GitHub Copilot to revitalize the look and feel of the site
  • Light and dark theme
  • Mobile responsive
  • Fairly large behind-the-scene customizations to original theme
  • GPT-4.1 AI generated Japanese and Singlish content
  • Site deployed and served via GitHub Pages
  • Home page
  • About Me page integrated with main site
  • Thanks page integrated with main site
  • Past Works page, in English original language content
  • Past Works page, in Japanese language content generated by GPT4.1
  • Past Works page, in Singlish language content generated by GPT4.1
