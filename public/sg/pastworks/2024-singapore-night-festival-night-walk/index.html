Singapore Night Festival 2024 - Night Walk

Year: 2024  |  corporate

This project make gamified web app for Singapore Night Festival 2024, can check in, redeem freebies. Work with creative and tech vendors, users, stakeholders, finish in short time. App engage participants for 9-day event, a lot of signups and check-ins.

  • Web app like Pokemon Go for Night Festival
  • Login > check-in > unlock audio stories > redeem rewards, all in one app
  • Gamified check-in screen for Singapore Night Festival 2024, everyone join in.
  • Audio story unlock interface for Night Festival app, listen and enjoy.
  • Reward redemption page for festival participants, collect reward steady.
  • User engagement statistics dashboard for Night Festival, see all the numbers.
  • Mobile responsive design showcase for Night Festival app, use on phone also can.
  • Collaborative development team interface for Night Festival app, teamwork steady.
